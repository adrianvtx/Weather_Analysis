Latitude vs. X


"What's the weather like as we approach the equator?" was the starting point for this data analysis project. Within given parameters, this project visualizes data for: maximum temperature, humidity, cloudiness, and windspeed; in relation to latitude per the geographic coordinate system. To accomplish this, we generated a list of over 500 latitude and longitude coordinates using a random number generator whose output was limited to the 'horizon' of the coordinate system ( -/+90 degrees). The coordinate list was passed to citipy, a Python library (https://pypi.python.org/pypi/citipy) which generated a new list of city names. To generate that new list, citipy recieved each coordinate pair and returned the name of the closest city. We eliminated duplicate city names from the list and created a query requesting data from the OpenWeatherMap API (https://openweathermap.org/api) to assemble a data set on the cities. After assembling the dataset, we used Matplotlib to plot each aspect of the weather vs. latitude. Weather factors we plotted per the limitations of the project were: temperature, humidity, cloudiness, and wind speed. We were further tasked to create a new website using HTML, css, and Bootstrap. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
